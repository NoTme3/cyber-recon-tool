# Cyber Recon Tool

A simple interactive tool for cybersecurity reconnaissance tasks including:

- **Subdomain enumeration** using public APIs  
- **Directory brute forcing** on subdomains with a customizable wordlist  
- Option to **save output** results to a file for later analysis

---

## Features

- Interactive menu-driven interface  
- Supports subdomain discovery for a target domain  
- Directory brute forcing on any given subdomain URL  
- Save results to text files for easy reference  
- Easy to extend and customize

---

## Requirements

- Python 3.x  
- `requests` library (`pip install requests`)
